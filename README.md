# Matt17r Tap

## How do I install these formulae?

`brew install matt17r/tap/<formula>`

Or `brew tap matt17r/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "matt17r/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
